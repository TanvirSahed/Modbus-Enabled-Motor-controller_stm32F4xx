# Universal Motor Controller (STM32F4xx)

A fully modular, dual-mode motor controller supporting both **DC** and **Stepper motors**, with
industrial-grade **Modbus TCP/RTU**, encoder-based feedback, PID control, and robust flash-backed configuration system.
